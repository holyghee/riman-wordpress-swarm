Kleines Test-Plugin, um Deployments zu verifizieren.

== Beschreibung ==

Dieses Plugin bietet einen Shortcode `[riman_test_plugin]`, der eine einfache OK‑Nachricht rendert. Damit lässt sich schnell prüfen, ob Code‑Deploys korrekt ankommen.

== Nutzung ==

Füge irgendwo im Inhalt den Shortcode ein:

    [riman_test_plugin]
